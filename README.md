## Port's One‑click experience

This package turns Port’s guides into a one‑click experience.

### Prerequisites
- Python 3.10+
- Port credentials (client credentials or API token)

### Install
```bash
pip install --upgrade port-experience
```

### Run
```bash
port-experience-apply
```

### Notes
- Set any required environment variables (e.g., Port credentials) in your shell or CI before running.
- See Port’s guide for background: https://docs.port.io/guides
